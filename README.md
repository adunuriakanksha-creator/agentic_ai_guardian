#Agentic Multi-modal Ai guardian 

Project: Agentic Multi-Modal AI Guardian

Team: ZENITH

Theme: AI & ML

#Overview:

We are building an autonomous personal safety app 
that detects danger automatically without the user 
pressing any button.

#Key features:

The app monitors 3 signals simultaneously:
1. Voice — detects distress keywords and stress
2. Motion — detects abnormal movement patterns  
3. GPS — detects unusual route deviations

#How it works:

When danger is detected, the Agentic AI automatically:
- Sends SOS to emergency contacts via SMS
- Shares live location
- Starts audio/video recording
- Finds nearest police station and hospital

#Tech Stack:

-Frontend:  Flutter (Mobile App)

-Backend:   Python + Flask

-Voice AI:  Google Speech API

-Location:  Google Maps API

-SMS Alert: Twilio
-Database:  Firebase
